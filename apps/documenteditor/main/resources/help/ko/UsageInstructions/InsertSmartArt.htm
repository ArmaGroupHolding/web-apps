SmartArt 삽입

SmartArt 그래픽은 가장 적합한 레이아웃을 선택하여 계층 구조의 시각적 표현을 만드는 데 사용됩니다.
ONLYOFFICE 스프레드시트 편집기는 타 사 편집기를 사용하여 삽입된 SmartArt 그래픽을 지원합니다. SmartArt가 포함된 파일을 열고 사용 가능한 편집 도구를 사용하여 그래픽 개체로 편집할 수 있습니다.
SmartArt 그래픽이나 그 요소를 클릭하면, 오른쪽 사이드바에서 도형서식이 활성화되어 레이아웃을 지정할 수 있습니다.

SmartArt 삽입하려면

  1. 삽입 탭으로 이동,
  2. SmartArt 버튼을 클릭,
  3. SmartArt 드롭다운 목록에서 유형을 선택,
  4. 유형별 SmartArt 갤러리에서 사용할 레이아웃을 선택합니다.

오른쪽 사이드바 도형서식에서 SmartArt 설정 변경이 가능합니다.

  • 채우기 - SmartArt 개체에 대한 채우기를 선택합니다.
    • 색상 채우기 - 선택한 SmartArt 개체의 내부 공간을 단색으로 지정합니다.
    • 그라데이션 채우기 - 그라데이션(Gradation)은 하나의 색채에서 다른 색채로 변하는 효과를 개체에 반영합니다.
      • 스타일 - 선형 또는 방사형 중에서 선택합니다.
        • 선형 - 색상이 왼쪽에서 오른쪽으로, 위에서 아래로 또는 단일 방향으로 선택한 각도로 색체가 변화됩니다. 방향 미리보기 창에 선택한 그라데이션 색상이 표시됩니다. 미리보기 창 화살표를 클릭하여 사전에 설정된 그라데이션 방향을 선택합니다. 각도는 정확한 그라데이션 방향을 위해 사용됩니다.
        • 방사형 - 개체에 원형으로 그라디언트를 적용하는 데 사용됩니다.
      • 그라데이션 중지점 - 한 색상에서 다른 색상으로 전환하기 위한 특정 지점입니다.
        • 그라데이션 중지점 추가 - 버튼 또는 슬라이더 막대를 사용하여 그라데이션 중지점을 추가합니다. 최대 10개의 그라데이션 중지점을 추가할 수 있습니다. 추가로 삽입되는 중지점은 현재 그라데이션 채우기 모양에 영향을 주지 않습니다.
        • 그라데이션 중지점 제거 - 그라데이션 중지점을 삭제합니다.
        • 위치 - 슬라이더 막대를 사용하여 그라데이션의 위치를 ​​변경하거나 백분율로 정확한 위치를 지정합니다.
        • 색 - 그라데이션 중지점 색상을 적용하려면 슬라이더 막대에서 중지점 버튼을 클릭한 다음 색을 클릭하여 원하는 색상을 선택합니다.
    • 그림 또는 질감 - SmartArt 개체 배경으로 그림 또는 미리 정의된 질감을 사용하여 개체 배경을 변경합니다.
      • 그림을 SmartArt 개체의 배경으로 사용하려면 현재 장치 (컴퓨터 하드 디스크 드라이브에서 선택), URL 경로 붙여 넣기 (외부 이미지 URL 경로를 입력), 스토리지 (공급사에서 제공한 저장소)에서 그림을 선택하여 배경에 그림을 적용합니다.
      • 질감을 SmartArt 개체의 배경으로 사용하려면 질감 메뉴에서 사전에 설정된 질감을 사용합니다. 현재 사용할 수 있는 질감은 갈색 종이,캔버스, 두꺼운 종이, 어두운 페브릭, 직물, 화강암, 회색 용지, 니트, 가죽, 파피루스,우드 입니다.
      • 선택한 그림의 크기가 SmartArt 개체보다 작거나 큰 경우 드롭다운 목록에서 늘이기 또는 타일을 선택할 수 있습니다. 늘이기는 SmartArt 개체 크기에 맞게 그림의 크기를 조정하여 공간을 완전히 채울 수 있습니다. 타일을 사용하면 개체보다 큰 그림인 경우 그림의 일부만 표시되며 개체보다 작은 경우는 공간을 완전히 채울 수 있도록 SmartArt 개체 표면 위에 반복된 그림으로 표시됩니다.
    • 패턴 - SmartArt 개체를 규칙적으로 반복되는 요소로 구성된 2색 디자인으로 배경을 적용합니다.
      • 패턴 - 메뉴에서 사전 정의된 디자인 중 하나를 선택합니다.
      • 전경색 - 패턴 요소의 색상을 적용합니다.
      • 배경색 - 패턴 배경색을 적용합니다.
    • 채우기 없음 - 채우기를 사용하지 않으려면 이 옵션을 선택하십시오.
  • 선 - SmartArt 개체의 외곽선의 너비, 색상, 대시 종류를 변경합니다.
    • 선 너비를 변경하려면 크기 드롭다운 목록에서 0.5pt, 1pt, 1.5pt, 2.25pt, 3pt, 4.5pt, 6pt 또는 선 없음을 선택합니다.
    • 선 색상을 변경하려면 아래 색상 상자를 클릭하여 색상을 선택합니다.
    • 대시 종류에서 선의 유형을 드롭다운 목록에서 선택합니다. (기본적으로 실 선이 적용되며 사용 가능한 파선 중 하나로 변경할 수 있음).
  • 음영 표시 - SmartArt 개체에 그림자를 적용합니다.

고급 설정을 클릭하여 자세한 설정을 조정할 수 있습니다.
